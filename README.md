# Zero Trust & Identity Lab 🔐

A hands-on lab demonstrating the transition from Traditional Perimeter Security to Zero Trust Architecture based on NIST SP 800-207.

## 🎯 Learning Objectives
- Implement identity-based connectivity with Tailscale
- Configure micro-segmentation using ACLs
- Apply least privilege with sudoers restrictions
- Use GenAI as a Security Co-pilot

## 📋 Prerequisites
- GitHub account
- Tailscale account (free)
- Kali Linux
- 2 hours of your time
